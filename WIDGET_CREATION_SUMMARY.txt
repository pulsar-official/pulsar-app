================================================================================
  KEY IMPLEMENTATION DETAILS
================================================================================

Data Integration:
  - Both widgets read from useProductivityStore
  - Use selector functions for habits and habitChecks arrays
  - Calculate "today" via dk(new Date()) function
  - Support optional props for testing/override

TypeScript:
  - Fully typed with @/types/productivity imports
  - Export interfaces for HabitChecklistProps and HabitChartWidgetProps
  - React.FC<Props> typing pattern used throughout

Styling:
  - OKLCH color system throughout (consistent with design system)
  - Purple accent: oklch(0.55 0.18 290)
  - Green success: oklch(0.65 0.14 150)
  - Dark background: oklch(0.09 0.02 270)
  - Follows CalendarWeekWidget styling patterns

Performance:
  - useMemo for expensive calculations (check maps, chart data)
  - ResizeObserver cleanup for memory management
  - No external dependencies (pure React + SVG)

Animations:
  - HabitChecklist: slideDown keyframe (0.4s) when habit completed
  - Smooth transitions on hover states
  - CSS animations (no Framer Motion required)

================================================================================
  HOW TO ADD TO CORESPACE DASHBOARD
================================================================================

File: src/components/Dashboard/modules/corespace/CorespaceLayout.tsx

Step 1: Add imports
  import HabitChecklist from '@/components/Dashboard/widgets/HabitChecklist'
  import HabitChartWidget from '@/components/Dashboard/widgets/HabitChartWidget'

Step 2: Add to AVAILABLE_WIDGETS array
  { id: 'habitChecklist',  name: 'Habits — Checklist',    tier: 'atom' },
  { id: 'habitChart',      name: 'Habits — Chart',        tier: 'molecule' },

Step 3: Add WidgetContainer instances in the widget grid
  <WidgetContainer
    id="habitChecklist"
    title="Habits Today"
    tier="atom"
    userTier={tier}
    defaultW={4}
    defaultH={5}
  >
    <HabitChecklist />
  </WidgetContainer>

  <WidgetContainer
    id="habitChart"
    title="Habit Trends"
    tier="molecule"
    userTier={tier}
    defaultW={6}
    defaultH={3}
  >
    <HabitChartWidget days={7} />
  </WidgetContainer>

================================================================================
  TESTING CONSIDERATIONS
================================================================================

Unit Testing:
  - Both components accept optional props for easy test setup
  - Can pass mock habits[] and habitChecks[] arrays
  - Can override onCheck/toggle behavior

Manual Testing:
  - Test with 0 habits (should show empty state)
  - Test with 1-5 habits (no scroll needed)
  - Test with 5+ habits (scroll bar should appear)
  - Test checking/unchecking habits
  - Test animation smoothness
  - Test responsive resizing for HabitChartWidget
  - Test dark theme contrast/readability

Edge Cases:
  - Empty habitChecks array (no checks recorded)
  - Future dates in habitChecks (filtered by <= TODAY)
  - Habits with special emoji characters
  - Habits with very long names (truncated with ellipsis)
